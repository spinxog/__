How to Test Argus

  1. Install the Argus extension (see README for setup)
  2. Click on any demo page below to open it
  3. Select text containing a claim (highlighted in yellow on demo pages)
  4. Click the Argus extension icon in your browser toolbar
  5. Click "Verify Claim" to see evidence analysis
  6. Click evidence cards to highlight source locations
