Argus Settings

Privacy First

Argus operates entirely locally by default. No data leaves your device unless you explicitly enable cloud fallback for deeper web searches.

Privacy & Security

Local-Only Mode
Process all claims locally without any cloud services. This ensures maximum privacy but may limit evidence discovery.
Encrypt Evidence Store
Password-protect your local evidence database. You'll need to enter a passphrase to access stored evidence.

Indexing & Performance

Auto-Index Pages
Automatically index visited pages for evidence when claims are verified. Improves future verification speed.

Cloud Fallback (Optional)

Enable Cloud Search
Allow Argus to search the broader web for additional evidence when local results are insufficient. Requires explicit opt-in for each search.

Note: Cloud fallback sends only canonicalized claims (no browsing history) and requires your explicit permission for each use.
